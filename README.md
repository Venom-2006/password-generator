# password-generator
🔐 Password Generator

A simple and secure Password Generator Web App built using React.
It allows users to generate strong passwords with customizable options such as length, numbers, and symbols to enhance security.

🚀 Features

1)Generate secure random passwords

2)Adjustable password length (8–30 characters)

3)Option to include numbers

4)Option to include special symbols

5)Copy password to clipboard instantly

6)Clean and simple user interface

🛠 Technologies Used

1)React

2)JavaScript (ES6)

3)HTML5

4)CSS3
